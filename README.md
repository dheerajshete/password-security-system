# Password Security System (C Language)

A simple C-based password security system built using string functions, random serial generation, and access verification logic.

## Features

* User name input
* Password validation (exactly 12 characters)
* Random serial number generation
* Unique passkey creation using:

  * reversed username
  * random serial number
* Passkey verification for access control

## Concepts Used

* `strlen()`
* `strcpy()`
* `strcat()`
* `strcmp()`
* `strrev()`
* `sprintf()`
* `rand()`
* `srand()`
* `time()`
* loops and conditional statements

## Purpose

This project was built to practice C string handling and logic building while creating a small real-world style security system.
